# Aqua Mirage Community Website

Versi statis siap GitHub Pages.

## Cara pakai
1. Upload semua file ini langsung ke root repository GitHub.
2. Buka Settings > Pages.
3. Source: Deploy from branch.
4. Branch: main, folder: /root.
5. Save.

Jangan upload folder luarnya saja. Root repo harus langsung berisi index.html, leaderboard.html, members.html, about.html, css, js, assets.

Discord invite sudah dipasang: https://discord.gg/kVqaRA6aqM
Firebase config sudah ada di js/firebase-config.js
